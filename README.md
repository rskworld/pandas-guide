Comprehensive guide to data manipulation with Pandas DataFrames including data cleaning, transformation, filtering, merging, and advanced operations.

## Description

This project provides a comprehensive guide to Pandas, the essential Python library for data manipulation. It covers DataFrame operations, data cleaning techniques, data transformation, filtering, merging, grouping, pivoting, and time series operations. Perfect for mastering data wrangling and preprocessing.

## Features

### Core Features
- DataFrame operations and indexing
- Data cleaning and preprocessing
- Data transformation and filtering
- Merging and joining datasets
- GroupBy and aggregation operations

### Advanced Features
- **Multi-Index Operations**: Working with hierarchical indexes
- **Window Functions**: Rolling, expanding, and exponentially weighted operations
- **Categorical Data**: Memory-efficient categorical variables
- **Advanced String Operations**: Regex, pattern matching, extraction
- **Large Dataset Handling**: Chunking and memory optimization
- **Data Validation**: Error handling and data quality checks
- **Advanced Indexing**: Custom indexes, at/iat for fast access
- **Export/Import Formats**: CSV, Excel, JSON, Parquet, HTML, SQL
- **Performance Optimization**: Vectorization, query optimization tips
- **SQL-like Operations**: JOIN, WHERE, GROUP BY, HAVING, ORDER BY
- **Time Series Operations**: Resampling, rolling windows
- **Pivot Tables and Cross Tabulation**: Data reshaping and aggregation

## Installation

1. Clone this repository or download the project files
2. Install the required dependencies:

```bash
pip install -r requirements.txt
```

## Usage

Open the Jupyter notebooks in the `notebooks/` directory to explore different topics:

- `01_dataframe_basics.ipynb` - Introduction to DataFrames
- `02_data_indexing.ipynb` - DataFrame indexing and selection
- `03_data_cleaning.ipynb` - Data cleaning and preprocessing
- `04_data_transformation.ipynb` - Data transformation techniques
- `05_filtering.ipynb` - Filtering and conditional operations
- `06_merging_joining.ipynb` - Merging and joining datasets
- `07_groupby_aggregation.ipynb` - GroupBy and aggregation operations
- `08_advanced_operations.ipynb` - Advanced Pandas operations including:
  - Multi-Index operations
  - Window functions and expanding operations
  - Categorical data handling
  - Advanced string operations with regex
  - Working with large datasets (chunking)
  - Data validation and error handling
  - Advanced indexing techniques
  - Export/import different formats
  - Performance optimization tips
  - SQL-like operations
  - Time series operations
  - Pivot tables and cross tabulation
