### üìö What's Included

#### 8 Comprehensive Jupyter Notebooks:
1. **01_dataframe_basics.ipynb** - Introduction to DataFrames
   - Creating DataFrames from various sources
   - Basic DataFrame operations
   - Column access and manipulation
   - Basic statistics

2. **02_data_indexing.ipynb** - DataFrame Indexing and Selection
   - Using loc[] for label-based indexing
   - Using iloc[] for integer-based indexing
   - Boolean indexing
   - Query() method

3. **03_data_cleaning.ipynb** - Data Cleaning and Preprocessing
   - Detecting missing values
   - Handling missing values (drop, fill, forward/backward fill)
   - Removing duplicates
   - Data type conversion

4. **04_data_transformation.ipynb** - Data Transformation
   - String operations
   - Applying functions
   - Date/time transformations
   - Data reshaping (pivot, melt)

5. **05_filtering.ipynb** - Data Filtering
   - Basic filtering with conditions
   - Multiple condition filtering
   - Using isin(), query()
   - Advanced filtering techniques

6. **06_merging_joining.ipynb** - Merging and Joining Datasets
   - Inner, Left, Right, Outer joins
   - Merging on different column names
   - Concatenating DataFrames
   - Using join() method

7. **07_groupby_aggregation.ipynb** - GroupBy and Aggregation
   - Basic GroupBy operations
   - Aggregation functions
   - Custom aggregation functions
   - Grouping by multiple columns
   - Transform and Apply

8. **08_advanced_operations.ipynb** - Advanced Pandas Operations
   - Pivot tables
   - Time series operations
   - Cross tabulation
   - Multi-Index operations
   - Window functions and expanding operations
   - Categorical data
   - Advanced string operations
   - Working with large datasets (chunking)
   - Data validation and error handling
   - Advanced indexing
   - Export/Import different formats
   - Performance optimization tips
   - SQL-like operations

### üöÄ Getting Started

1. Clone the repository:
```bash
git clone https://github.com/rskworld/pandas-guide.git
cd pandas-guide
```

2. Install dependencies:
```bash
pip install -r requirements.txt
```

3. Open Jupyter Notebook:
```bash
jupyter notebook
```

4. Start with `01_dataframe_basics.ipynb`
