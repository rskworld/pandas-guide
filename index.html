This project provides a comprehensive guide to Pandas, the essential Python library for data manipulation. Perfect for mastering data wrangling and preprocessing.

ğŸ“– Notebooks

  • 01_dataframe_basics.ipynb Introduction to Pandas DataFrames - creating DataFrames, understanding structure, and basic operations.
  • 02_data_indexing.ipynb DataFrame indexing and selection - using loc, iloc, and other selection techniques.
  • 03_data_cleaning.ipynb Data cleaning and preprocessing - handling missing values, removing duplicates, and data type conversion.
  • 04_data_transformation.ipynb Data transformation techniques - applying functions, string operations, and data reshaping.
  • 05_filtering.ipynb Data filtering - conditional filtering, query() method, and advanced filtering techniques.
  • 06_merging_joining.ipynb Merging and joining datasets - inner, left, right, outer joins, and concatenation.
  • 07_groupby_aggregation.ipynb GroupBy and aggregation operations - grouping data and performing aggregations.
  • 08_advanced_operations.ipynb Advanced Pandas operations - pivot tables, time series, and cross tabulation.
